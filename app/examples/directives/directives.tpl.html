Directives Example

Description:

This example creates a two directives simpleDirective and controllerDirective.

The results of this directives can be seen below.

Simple Directive

The simple-directive just display a template without any further actions.

Directive with Controller

The controller-directive has an own controller, like a view-controller. Beside this the directive exchanges data between itself and the parent view. Just type something in the input field below ...

app/examples/directives/

Files:

  • directives.route.js - template routing
  • directives.tpl.html - content template

  • directives.simple.directive.tpl.html - simple directive template
  • directives.controller.directive.js - controller directive creation
